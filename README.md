# UVA26 Sailplane Capstone

This repository contains files for our capstone project analyzing IGC flight logs from motorized sailplanes.

## Project Goal

The goal of this project is to evaluate pilot behavior during in-flight engine starts and determine whether pilots are following manufacturer-recommended procedures.
